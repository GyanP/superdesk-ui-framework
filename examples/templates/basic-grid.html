Basic grid

The basic grid in Superdesk is based on flex grid and works very similarly to the standard float grid. It is intended for positioning items in a grid layout, either on separate pages or inside other elemts. Basic responsive features are included but features needed for a full layout grid are not available as Superdesk doesn't need them at this point. A full featured layout grid might be added as at later point as a separate module.

Note: flex grid is only supported in Chrome, Firefox, Safari 6+ and IE10+.
