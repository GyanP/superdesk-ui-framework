Layout grid

Layout grid in Superdesk is based on flex grid and works very similarly to the standard float grid. It has 12 columns setup, and column gutter of 10px so that the space between two columns is always 20px. The grid is fluid, but for now we don't have responsive behavior in terms of setting different number of columns for various screen sizes. It's because in Superdesk, layout grid will mainly be used inside other containers or components (like panes, modals etc.) so media queries do not make much sense.

If there is a need to have layout grid inside an element which already has default padding (like modal body for instance), just add modifier .grid--padding-fix to div .grid.

Examples

Add grid container with class .grid, and nest grid items inside. For example, grid item will be .grid__item with additional modifier class .grid__item--col-3, so number of columns is set as a modifier on a grid item div. To have a full row width, sum of items columns should be 12.
