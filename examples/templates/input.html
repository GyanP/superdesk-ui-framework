Inputs

Line inputs are based on Material design style inputs. They include useful elements like hint text .sd-line-input__hint, error messages .sd-line-input__message and character count .sd-line-input__char-count.

Example Markup

// Input with hint text and character count

00/90
This is a hint text. Duis autem vel eum iriure dolor in hendrerit in vulputate velit esse molestie consequat.

// Required input field

Hendrerit in vulputate velit esse molestie consequat.

// Input with error message

This field is required.

// Disabled input

This field is disabled.

// Input with no dedicated space for the hint text at the bottom

// File input

// Alternative styling

An alternative styling for input fields is available in cases where the whole filed needs to be visible. The style is applied by appending .sd-line-input--boxed to the default classes.
