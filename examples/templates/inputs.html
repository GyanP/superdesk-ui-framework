Inputs

Line inputs are based on Material design style inputs. They include useful elements like hint text .sd-line-input__hint, error messages .sd-line-input__message and character count .sd-line-input__char-count.

Example Markup

// Input with hint text and character count.

00/90
This is a hint text. Duis autem vel eum iriure dolor in hendrerit in vulputate velit esse molestie consequat.

// Input with error message.

This field is required.

// Inline input fields.

Sed a nunc non lectus fringilla suscipit.
Proin ornare ligula eu tellus tempus elementum.
In facilisis scelerisque dui vel dignissim.
Submit

// Inline input fields with additional containers.

Duis autem vel eum iriure.
Dolor in hendrerit.
Duis autem vel dolor.
Velit esse consequat.
Iriure dolor in hendrerit.
Submit
